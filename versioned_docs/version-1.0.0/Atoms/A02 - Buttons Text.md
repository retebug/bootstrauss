## Description

Buttons perform actions. In our apps the hierarchy of importance is emphasized and delineated by using primary, secondary and tertiary buttons.

## Primary Button

### Default

<a class="btn btn--primary">
  Primary
</a>
<a class="btn btn--primary btn--disabled">
  Primary Disabled
</a>
